# TASK 3 : Portfolio Personnel 💼

Un site web vitrine premium pour présenter vos compétences et projets (dont les 3 autres tâches CodeAlpha).

## ✨ Fonctionnalités
- **Design Glassmorphism Premium :** Panneaux transparents givrés sur fond coloré.
- **Arrière-plan Animé :** Des formes lumineuses (Blobs) flottent continuellement en fond pour donner de la profondeur à l'UI.
- **Effet de Frappe Automatique :** Le sous-titre de l'écran d'accueil s'écrit et s'efface dynamiquement.
- **Curseur Personnalisé :** Remplacement du curseur classique par un anneau lumineux qui suit la souris de manière fluide et réagit au survol des liens.
- **Cartes 3D :** La section projets utilise un effet de Tilt 3D.
- **Routage Interne :** Les cartes projets mènent directement aux fichiers locaux des Tâches 1, 2 et 4.

## 🛠️ Lancement
Ouvrez le fichier `index.html` dans un navigateur web. Idéal pour être déployé sur GitHub Pages ou Netlify !
